# DESIRE.co

## UAS Pemrograman Web

## Deskripsi Proyek

DESIRE.co merupakan website e-commerce sederhana yang dibuat sebagai proyek Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web. Website ini digunakan untuk menampilkan koleksi T-shirt dengan desain original, memungkinkan pengguna melihat detail produk, menambahkan produk ke keranjang, melakukan checkout, memilih metode pembayaran, hingga mengirim konfirmasi pesanan melalui WhatsApp.

---

# Business Documentation

## Latar Belakang

Perkembangan bisnis fashion semakin pesat sehingga dibutuhkan media penjualan yang mudah diakses secara online. DESIRE.co dibuat sebagai website sederhana yang memudahkan pelanggan melihat koleksi T-shirt serta melakukan pemesanan secara praktis.

## Tujuan

- Memenuhi tugas UAS Mata Kuliah Pemrograman Web.
- Membuat website e-commerce sederhana.
- Memberikan pengalaman berbelanja secara online.
- Menerapkan HTML, CSS, Bootstrap, dan JavaScript dalam pembuatan website.

## Target Pengguna

- Remaja
- Mahasiswa
- Pecinta fashion
- Pengguna yang ingin membeli T-shirt secara online.

# Cara Menggunakan Website

1. Membuka halaman utama.
2. Melihat koleksi T-shirt.
3. Klik **Lihat Detail** untuk melihat informasi produk.
4. Tambahkan produk ke keranjang.
5. Klik ikon keranjang.
6. Pilih **Checkout**.
7. Isi nama, nomor WhatsApp, dan alamat.
8. Pilih metode pembayaran.
9. Konfirmasi pembayaran.
10. Kirim konfirmasi melalui WhatsApp.
